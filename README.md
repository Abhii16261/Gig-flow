# GigFlow (Full Stack)

This repo contains:
- `frontend/` Vite + React UI
- `backend/` Node + Express API

## Run locally

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

## Deploy
- Backend: Render (rootDir: backend)
- Frontend: Vercel (rootDir: frontend)
